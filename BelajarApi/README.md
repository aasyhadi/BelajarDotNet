# BelajarApi 

Backend REST API menggunakan ASP.NET Core 9 dan SQL Server.

## Features

* JWT Authentication
* Refresh Token
* Role Based Authorization
* Repository Pattern
* Service Layer
* DTO Pattern
* AutoMapper
* FluentValidation
* Audit Log
* Dashboard Analytics
* File Upload
* KRS Management

## Database

```bash
dotnet ef database update
```

## Run Application

```bash
dotnet restore
dotnet run
```

Default URL:

```text
https://localhost:7xxx
http://localhost:5xxx
```

## Testing

```bash
dotnet test
```

## Coverage

```bash
dotnet test --collect:"XPlat Code Coverage"
```
